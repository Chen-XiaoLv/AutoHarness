## Learned Rules

1. **Answer Concisely and Format Consistently**: When answering questions, output only the most direct and concise answer, prioritizing words or short phrases. Ensure the answer format matches the standard answer exactly, including case, punctuation, abbreviations, name parts (e.g., using surname only if appropriate), and term forms (e.g., singular vs. plural). Include required articles (e.g., 'a', 'an') and integral parenthetical details as per common usage. Use the most concise standard form unless the question explicitly requests more detail, and avoid adding any extra modifications, variants, or redundant text.

2. **Identify Answer Type**: First, identify the type of answer required (e.g., work title, person's name, conceptual term). Then, output the corresponding entity directly, without confusing related elements or adding irrelevant details. For person names, prefer the surname or common name unless the question explicitly uses the full name; for terms, use the form as it appears in the question or the standard singular form, choosing the most concise and standard form based on common usage. Ensure that the answer granularity matches the question context by retaining all essential identifying elements (e.g., full name parts, parenthetical details, or specific descriptors) that define the entity, unless the question explicitly simplifies the request.
