# Question Answering Skill

## Learned Rules

### 规则1：处理列表与填空类问题
当问题呈现一个列表（如“X, Y, Z”）或填空格式（如“A ___”）时，答案通常不是列表本身或填空处，而是这些元素的共同类别、特定名称或缺失的词。必须分析上下文，提取最具体、最直接的答案（例如，品牌列表的类别、填空的正确词语）。

### 规则2：提取最简短的精确答案
在所有问答任务中，答案必须是从上下文或问题中直接提取的、支持精确匹配的、最简短的名词性短语。对于人名、品牌、地名等具体实体，必须直接输出该实体名称，禁止输出其类别、角色、描述、翻译或同义转述。答案的形式（包括大小写、拼写、词性和连字符等）必须与上下文中提供线索的原词或标准答案格式完全一致。

**精确匹配与完整提取**：答案必须与标准答案在词序、拼写、大小写、标点甚至括号内容上完全一致。绝对禁止将标准答案替换为任何同义词、近义词、描述性短语或重新表述。如果标准答案包含括号说明或修饰词（如‘Pseudopod (the false pod, false foot)’），必须完整输出。同时，答案必须是从上下文中提取一个完整的、不可再分割的名词性短语片段。禁止在提取的原词上添加任何修饰词（如形容词、所有格‘his/her’）、类别后缀（如‘City’、‘family’）或进行任何解释性扩展。也禁止为了追求简短而截断必要的限定成分（如从‘his hat’中截取出‘hat’）。答案应完整反映上下文中支持标准答案的最小必要表述。
